2.1. Einstellungen¶

Ab Version 2.5 wird keine weitere Installation für Windows erforderlich. Laden Sie einfach das Archiv herunter, entpacken Sie es und starten Sie die UncertRadio.exe-Datei im Verzeichnis /bin.

2.1.1.1. Länderspezifische Parameter¶

Die Parameter in der Konfigurationsdatei UR2_cfg.dat unter dem Abschnitt [Lokal] können wie folgt definiert werden.

  1. die Parameterwerte nach dem „=“-Zeichen können weggelassen werden:

    Decimal_point=

    List_separator=

    Language=

    Damit werden die Werte für das Dezimaltrennzeichen und das Listen-Trennzeichen (für CSV-Dateien) ausgewählt:

    DE oder FR: Komma als Dezimaltrennzeichen; Semikolon als Listen-Trennzeichen

    EN: Punkt als Dezimaltrennzeichen; Komma als Listen-Trennzeichen

    Hinweis: Die Sprache, die im Hauptmenü - Optionen ausgewählt werden kann, wird nur vorübergehend verwendet, d.h. nur innerhalb des aktuellen Programmlaufs.

  2. Wenn andere Spezifikationen gewünscht sind, können ihre Parameterwerte direkt nach dem „=“-Zeichen in der Konfigurationsdatei eingetragen werden. Es kann ausreichen, nur das gewünschte Listen-Trennzeichen explizit einzufügen:

    List_separator=;

  3. Die Sprache kann auch im Dialog des Menüpunkts „Optionen – Vor-Einstellungen“ nachdem das Programm bereits gestartet wurde, ausgewählt werden (aber nur vorübergehend; siehe oben). Aus der dort definierten Sprache werden die Zeichen für Dezimaltrennzeichen und Listen-Trennzeichen wie oben unter a) beschrieben bestimmt. Zudem kann das Listen-Trennzeichen dort explizit ausgewählt werden.

  4. Für eine grafische Darstellung wird das Dezimaltrennzeichen durch die Sprachkürzel DE, EN oder FR bestimmt

2.1.1.2. Sprachauswahl¶

Der Nutzer von UncertRadio kann durch das Programm in verschiedenen Dialogen auf Deutsch, Englisch und Französisch geführt werden. Der Nutzer kann zwischen Deutsch, Englisch und Französisch wählen. Dies wird über den Eintrag ‚Sprache=‘ in der oben genannten Konfigurationsdatei UR2_cfg.dat gesteuert, die bei jedem Programmstart gelesen wird. Die Parameterwerte sind:

  • DE für Deutsch,

  • EN für Englisch,

  • FR für Französisch.

Die Sprache kann im laufenden Programm über den Dialog „Menü – Optionen – Vor-Einstellungen“ gewechselt werden. Seit Version 2.3.07 umfasst dies auch die „GTK Standard-Buttons“, die durch ein Symbol und Text gekennzeichnet sind und die zuvor erst nach einem neuen Programmstart übersetzt wurden. Dies wurde erreicht, indem alle „Standard-Buttons“ in normale Buttons umgewandelt wurden; die Form der Button-Symbole hat sich nur leicht verändert. Für das komplexere GTK-Dateiauswahldialogfeld sind die meisten Elemente übersetzt, aber nicht alle.

2.1.2. Schriftart und Schriftgröße¶

UR erlaubt die Änderung der Schriftart und der Schriftgröße. Die Einstellungen werden in der Settings.ini-Datei gespeichert:

[Settings]
gtk-theme-name = win64
gtk-font-name = Sans Normal 10

Beachten Sie, dass die Zeichenketten links von den =-Zeichen nicht geändert werden dürfen. Beim Start des UR-Programms wird die Settings.ini-Datei geladen. Diese Datei sollte Teil des UncertRadio.exe-Pfads sein.

Zur Änderung der Werte für gtk-font-name kann das Schriftname-Symbol der Symbolleiste verwendet werden.

Mit dem Schriftname-Symbol kann die Schriftart und/oder die Schriftgröße leicht geändert werden. Es muss berücksichtigt werden, dass durch die Vergrößerung der Schriftgröße auch das Fenster des Programms größer wird.

Mittels des Dialogbuttons „Anwenden„ wird die gewählte Schriftart auf das UR-Fenster angewendet. Wenn die Schriftart als akzeptabel angesehen wird, kann sie mit dem Dialogbutton „Speichern„ in der Datei Settings.ini gespeichert werden; dies erfordert jedoch, dass die Schriftart einmal auf dem Programmfenster angewendet wurde. Die neue Schriftart wird nicht gespeichert, wenn der Dialog stattdessen mit dem Button „Beenden„ geschlossen wird.

2.1.3. Verknüpfung von Projektdateien (.txp) mit UncertRadio.exe¶

Die Verknüpfung des Dateityps .txp mit UncertRadio.exe kann in Windows folgendermaßen hergestellt werden:

  • Markieren Sie eine .txp-Datei, klicken Sie mit der linken Maustaste auf „mit dem folgenden Programm öffnen“ und wählen Sie „anderes Programm“

  • Setzen Sie das Häkchen und klicken Sie auf „Weitere Programme“

  • Scrollen Sie die Liste der empfohlenen Programme nach unten

  • Klicken Sie auf „Weitere installierte Programme auf diesem PC suchen“

  • Wählen Sie UncertRadio.exe aus

Beim nächsten Doppelterklick auf den Dateinamen der .txp-Datei sollte das Programm erfolgreich starten.

2.1.4. Excel-gesteuerte Verarbeitung von UncertRadio¶

Die Excel-Datei UR2_SingleAutoRun_V12.xlsm wurde im Vergleich zur Datei der vorherigen Version geändert, indem die Dateinamen so erweitert wurden, dass sie immer den vollständigen Pfad beinhalten. Ein weiterer Auswertungsbutton „Start (autosep)“ wurde hinzugefügt; siehe Abschnitt 3.2. Die Excel-Datei kann in den 32-Bit- und 64-Bit-Versionen von MS Office/Excel verwendet werden; siehe die Hinweise im Einführungsteil des VBA-Moduls (Modul_Auto_single_UR) in der Excel-Datei.

Das VBA-Modul der Excel-Datei interpretiert jetzt einen UncertRadio-Beendigungscode, der mit Version 2.2.03 eingeführt wurde. Ein Beendigungscode von 3 (Fehler aufgetreten während des UncertRadio-Laufs) führt zu einem Abbruch des laufenden VBA-Moduls.
